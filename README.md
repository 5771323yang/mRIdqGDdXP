# 财务管理系统简介 java198

## 项目概述

本项目是一个基于Springboot和Vue的前后端分离的财务管理系统。采用经典的MVC三层架构模式，结合Spring、SpringBoot、Mybatis和Vue等技术开发，实现了Web形式的业务处理系统。系统界面友好，操作便捷。

## 技术栈

- **前端**：Vue.js
- **后端**：Springboot、Javaee
- **数据库**：MySQL
- **构建工具**：Maven

## 功能模块

1. 财务管理
   - 账目录入
   - 账目查询
   - 账目统计
2. 用户权限管理
   - 用户注册
   - 用户登录
   - 权限控制

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- **操作系统**：Windows/Linux/MacOS
- **数据库**：MySQL 5.7+
- **Java版本**：JDK 1.8+

## 部署步骤

1. 准备环境
   - 安装Java开发环境
   - 安装MySQL数据库
   - 部署前后端服务
2. 导入数据库
   - 使用提供的数据库脚本初始化数据库
3. 启动服务
   - 启动后端Springboot服务
   - 启动前端Vue服务

## 核心亮点

- **前后端分离**：使得项目结构更加清晰，便于管理。
- **灵活的权限管理**：根据不同用户角色分配不同权限，确保系统安全。
